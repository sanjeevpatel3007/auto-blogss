## 📌 Project Overview

**Auto AI Blog Generator** is a zero-maintenance blogging engine that:

- ⏱️ **Runs every 10 minutes** using GitHub Actions cron
- 🔍 **Fetches trending topics** via Google Trends (with a smart fallback list)
- ✍️ **Generates a short blog post** (100–200 words) using the **Kimi-K2** model via OpenRouter
- 💾 **Saves each post as a Markdown file** inside the `/blogs` folder
- 🔁 **Auto-commits and pushes** directly to your repository
- 🚫 **No database, no server, no hosting cost** — just GitHub

---

## ⚙️ How It Works

```
Every 10 minutes
      │
      ▼
GitHub Actions triggers the workflow
      │
      ▼
Python script fetches a trending topic
(Google Trends → fallback list if unavailable)
      │
      ▼
Builds an AI prompt and calls OpenRouter API
(Model: moonshotai/kimi-k2)
      │
      ▼
Parses the AI response → Title + Body
      │
      ▼
Saves as /blogs/<title-slug>.md
      │
      ▼
Git commit + push → new file appears in the repo
```

## 🚀 Setup Steps

### 1. Fork or Clone This Repository

```bash
# Fork via GitHub UI, then clone your fork:
git clone https://github.com/<your-username>/auto-ai-blog-generator.git
cd auto-ai-blog-generator
```

### 2. Get Your OpenRouter API Key

1. Go to [openrouter.ai](https://openrouter.ai) and create a free account
2. Navigate to **API Keys** → click **Create Key**
3. Copy the key (starts with `sk-or-...`)

### 3. Add the API Key as a GitHub Secret

1. Open your forked repo on GitHub
2. Go to **Settings** → **Secrets and variables** → **Actions**
3. Click **New repository secret**
4. Name: `OPENROUTER_API_KEY`
5. Value: paste your key
6. Click **Add secret**

### 4. Enable GitHub Actions

1. Go to the **Actions** tab of your repo
2. If prompted, click **"I understand my workflows, go ahead and enable them"**
3. The workflow will now run automatically every 10 minutes

### 5. Test It Manually (Optional)

1. Go to **Actions** → **Auto AI Blog Generator**
2. Click **Run workflow** → **Run workflow**
3. Watch the run complete and check the `/blogs` folder for a new `.md` file

## ✏️ How to Customize Topics

Open `scripts/generate_blog.py` and edit the `FALLBACK_TOPICS` list:

```python
FALLBACK_TOPICS = [
    "Artificial Intelligence in Healthcare",
    "Quantum Computing Breakthroughs",
    "Your Custom Topic Here",          # ← add yours
    ...
]
```

These topics are used when:
- `pytrends` (Google Trends) is unavailable
- All live trending topics are already covered

You can also change the AI model by editing the `MODEL` constant:

```python
MODEL = "moonshotai/kimi-k2:free"   # swap for any OpenRouter model
```

## 🔧 Running Locally

```bash
# Install dependencies
pip install -r requirements.txt

# Set your API key
export OPENROUTER_API_KEY="sk-or-your-key-here"

# Run the generator
python scripts/generate_blog.py
```

A new `.md` file will appear inside `/blogs/`.
